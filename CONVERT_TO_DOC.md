# Converting Manual Testing Protocol to .doc File

## Method 1: Using Microsoft Word (Recommended)

1. Open Microsoft Word
2. Go to **File → Open**
3. Select `MANUAL_TESTING_PROTOCOL_TABLE.md`
4. Word should recognize the markdown format
5. Save as **Word Document (.docx)**

## Method 2: Using Pandoc (If Installed)

```bash
pandoc MANUAL_TESTING_PROTOCOL_TABLE.md -o MANUAL_TESTING_PROTOCOL.doc
```

## Method 3: Copy-Paste Method

1. Open `MANUAL_TESTING_PROTOCOL_TABLE.md` in any text editor
2. Select All (Ctrl+A)
3. Copy (Ctrl+C)
4. Open Microsoft Word
5. Paste (Ctrl+V)
6. Word will preserve the table formatting
7. Save as .doc or .docx

## Method 4: Online Converter

Use an online Markdown to DOCX converter:
- https://www.markdowntopdf.com/
- https://www.zamzar.com/convert/md-to-docx/
