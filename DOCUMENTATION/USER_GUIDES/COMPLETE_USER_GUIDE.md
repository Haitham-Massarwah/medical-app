## Getting Started

### First Time Setup

1. **Download the Application**
   - Desktop: Download `.exe` file
   - Web: Visit your clinic's portal
   - Mobile: Install from app store (coming soon)

2. **Create an Account**
   - Click "Register"
   - Enter your email and password
   - Verify email address
   - Complete profile information

3. **Login**
   - Enter your credentials
   - System detects your role automatically
   - You're ready to use the system!

## For Patients

### Booking an Appointment

1. **Find Your Doctor**
   - Browse by specialty
   - Search by name
   - View ratings and reviews

2. **Select Time Slot**
   - View doctor's availability
   - Choose date and time
   - Add to cart (if booking multiple)

3. **Complete Booking**
   - Review details
   - Enter payment information
   - Confirm booking
   - Receive confirmation email/SMS

### Managing Appointments

**View Appointments:**
- Go to "My Appointments"
- See all scheduled, past, and upcoming appointments
- Filter by status, date, or doctor

**Reschedule:**
1. Open appointment details
2. Click "Reschedule"
3. Select new date/time
4. Confirm changes

**Cancel:**
1. Open appointment
2. Click "Cancel"
3. Confirm cancellation
4. Check refund policy

### Using Waze Navigation

1. Open appointment details
2. Click Waze icon
3. Navigation opens automatically
4. Follow route to clinic

## Troubleshooting

### Common Issues

**Problem: Can't login**
- Check email/password
- Verify internet connection
- Try password reset
- Contact support

**Problem: Appointment not showing**
- Check filters
- Refresh page
- Clear cache
- Check timezone

**Problem: Payment failed**
- Verify card details
- Check card balance
- Try different payment method
- Contact support

**Problem: Notifications not received**
- Check spam folder
- Verify email/SMS settings
- Enable notifications in app settings

## FAQ

**Q: How do I cancel an appointment?**  
A: Open the appointment, click "Cancel", confirm, check refund policy.

**Q: Can I book multiple appointments at once?**  
A: Yes! Add appointments to cart and checkout together.

**Q: What if I need to reschedule?**  
A: Open appointment, click "Reschedule", select new time.

**Q: How do I find my doctor's location?**  
A: Click Waze icon on appointment details for navigation.

**Q: Do I need to create an account?**  
A: Yes, account required for security and appointment tracking.

**Q: How secure is my information?**  
A: All data encrypted, HIPAA-compliant architecture, 2FA available.

**Q: Can I use the app on mobile?**  
A: Currently desktop and web. Mobile app coming soon.
