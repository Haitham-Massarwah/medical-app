## 🚀 **HOW TO RUN**

### Option 1: Quick Test (Recommended First)
```bash
# Double-click
TEST.bat

# Or run
flutter run -d chrome
```

### Option 2: Full System with Docker
```bash
# Start all services
docker-compose up -d

# Check status
docker-compose ps

# View logs
docker-compose logs -f

# Stop services
docker-compose down
```

### Option 3: Production Deployment
```bash
# Build and deploy
kubectl apply -f k8s/

# Check deployment
kubectl get pods

# Scale up
kubectl scale deployment medical-appointments-backend --replicas=5
```
