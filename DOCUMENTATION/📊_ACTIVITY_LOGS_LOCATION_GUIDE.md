# 📊 Activity Logs Location Guide

## 🎯 **Where to Find the Comprehensive Activity Logs**

The comprehensive activity logs are located in the **System Logs** page, which can be accessed through multiple paths:

### **Path 1: Developer Role → System Logs**
1. **Open the app** and select **"Developer"** role
2. **Click "יומני מערכת" (System Logs)** card
3. **View the comprehensive logging dashboard**

### **Path 2: Security Dashboard → System Logs**
1. **Open the app** and select **"Developer"** role
2. **Click "לוח אבטחה" (Security Dashboard)** card
3. **Click "יומני מערכת" (System Logs)** button
4. **View the comprehensive logging dashboard**

## 📥 **Download Options Available**

### **📊 Bulk Download (All Logs)**
- **CSV Format**: Spreadsheet-compatible data
- **JSON Format**: Structured data for APIs
- **TXT Format**: Human-readable logs

### **📅 Daily Logs (Separate Files Per Day)**
- **Calendar Icon**: Click the calendar icon (📅) in the app bar
- **Daily Files**: Creates separate files for each day
- **3 Formats Per Day**: CSV, JSON, TXT for each day
- **File Naming**: `YYYY-MM-DD_system_logs.format`

### **📄 Individual Log Downloads**
- **Download Button**: Orange download button for each log
- **Detailed TXT Files**: Complete context and metadata
- **Formatted Content**: Human-readable log details

### **Step 1: Launch the App**
The app now starts with a **Login Page** that includes:
- **Customer Registration**: Only customers can register
- **Developer Login**: Special login for developers
- **Role Selection**: Clear indication of user types

### **Step 2: Login as Developer**
- **Email**: `developer@medicalapp.com`
- **Password**: Any password (simulated)
- **Or click**: "התחבר כמפתח" (Login as Developer)

### **Step 3: Access System Logs**
- **Main Page**: Click "יומני מערכת" (System Logs)
- **Security Dashboard**: Click "לוח אבטחה" → "יומני מערכת"

### **Step 4: Explore the Logs**
- **View Analysis**: See comprehensive statistics
- **Browse Logs**: Scroll through 40+ detailed log entries
- **Filter Logs**: Use filters by level, date, category
- **Download Logs**: Use bulk or daily download options
