# 📋 Developer Doctor Management Guide

## 🎯 **How Developers Add Doctors/Therapists**

The developer can add doctors and therapists through the **User Management** system, which is already fully functional!

### **📍 Access Path:**
1. **Login as Developer** (use developer login button)
2. **Click "ניהול משתמשים" (User Management)** card
3. **Select "רופאים/מטפלים" (Doctors/Therapists)** tab
4. **Click "הוסף רופא/מטפל" (Add Doctor/Therapist)** button

### **🔧 Add Doctor/Therapist Features:**

#### **📝 Doctor Registration Form:**
- **Full Name**: Doctor's complete name
- **Email**: Professional email address
- **Phone**: Contact number
- **Specialty**: Medical specialty (dropdown selection)
- **License Number**: Medical license ID
- **Password**: Secure password for the doctor
- **Status**: Active/Inactive/Blocked
- **Join Date**: Registration date
- **Additional Info**: Notes and specializations

#### **🎛️ Management Capabilities:**
- **Add New Doctors**: Complete registration form
- **Edit Existing**: Modify doctor information
- **Block/Unblock**: Control access status
- **Remove Doctors**: Delete from system
- **View Details**: Complete doctor profile
- **Search & Filter**: Find specific doctors
- **Bulk Operations**: Manage multiple doctors

### **👥 Customer Registration (Restricted)**
- **Only customers can register** through the login screen
- **Doctors are added by developers only**
- **Clear role separation** maintained
