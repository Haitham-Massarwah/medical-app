# 💰 INSTALLATION & BUYING GUIDE

## 📥 INSTALLING THE DESKTOP APPLICATION

### Option 1: Standalone Executable (Recommended)

**Download & Install:**
1. Download `medical_appointment_system.exe`
2. Download `data` folder (if separate)
3. Place both in a folder (e.g., `C:\Program Files\MedicalApp\`)
4. Double-click `.exe` to launch
5. No installation wizard needed!

**Benefits:**
- ✅ No registry changes
- ✅ Portable (can move folder)
- ✅ No admin rights needed
- ✅ Easy to uninstall (delete folder)
