## 📱 **ANDROID PHONE/TABLET**

### Method 1: Google Play Store (Recommended)

**Steps:**
1. Open **Google Play Store** app on your Android device
2. Tap the search bar at the top
3. Type: **"Medical Appointment System"**
4. Tap the app when it appears
5. Tap **"Install"**
6. Wait for download (30-50 MB)
7. Tap **"Open"** when done
8. App is installed!

**Find the app:**
- App drawer (list of all apps)
- Home screen (if you added shortcut)
- Just search "Medical" in app drawer

### Method 2: Direct APK Install (For Testing)

**If the app is not on Play Store yet:**

1. **Enable Unknown Sources:**
   - Open **Settings**
   - Go to **Security** or **Privacy**
   - Enable **"Unknown sources"** or **"Install unknown apps"**
   - Allow for your browser or file manager

2. **Download APK:**
   - Open browser on phone
   - Visit download link provided by clinic
   - Download `MedicalAppointments.apk`

3. **Install:**
   - Tap the downloaded file
   - Tap **"Install"**
   - Wait for installation
   - Tap **"Open"**

4. **Done!**
   - App is installed
   - Find it in app drawer

**File Size:** ~30-50 MB  
**Requires:** Android 5.0 or higher

## 💻 **WINDOWS COMPUTER**

### Method 1: Download Installer (Easiest)

**Steps:**
1. **Download:**
   - Visit: `https://your-clinic.com/download`
   - Click **"Download for Windows"**
   - Save `MedicalAppointments-Setup.exe`

2. **Run Installer:**
   - Double-click the downloaded file
   - If Windows SmartScreen appears, click **"More info"** → **"Run anyway"**
   - Click **"Next"** through the wizard
   - Choose installation folder (default is fine)
   - Click **"Install"**
   - Wait 30 seconds
   - Click **"Finish"**

3. **Launch:**
   - Start menu → Find "Medical Appointments"
   - Double-click to open
   - Or use desktop shortcut

4. **Done!**
   - App is installed in: `C:\Program Files\MedicalAppointments\`

### Method 2: Portable Version (No Installation)

**Steps:**
1. Download ZIP file
2. Extract to any folder
3. Run `medical_appointment_system.exe`
4. No installation needed!

**File Size:** ~50-80 MB  
**Requires:** Windows 10 or higher

## 🍎 **MAC COMPUTER**

### Method 1: Download DMG (Easiest)

**Steps:**
1. **Download:**
   - Visit: `https://your-clinic.com/download`
   - Click **"Download for Mac"**
   - Save `MedicalAppointments.dmg`

2. **Install:**
   - Double-click the downloaded DMG file
   - A window opens with app icon
   - Drag the app to the **Applications** folder
   - Eject the DMG (optional)

3. **First Launch:**
   - Open **Applications** folder
   - Find "Medical Appointment System"
   - Right-click → **"Open"** (first time only)
   - Click **"Open"** in security dialog
   - Next times: just double-click normally

4. **Done!**
   - App is in Applications folder
   - Add to Dock for quick access

### Method 2: Mac App Store

**Steps:**
1. Open **App Store**
2. Search "Medical Appointment System"
3. Click **"Get"**
4. Enter Apple ID password
5. App downloads and installs
6. Find in Applications folder or Launchpad

**File Size:** ~40-70 MB  
**Requires:** macOS 10.15 (Catalina) or higher

## 🆘 **COMMON INSTALLATION ISSUES**

### Android

**Issue:** "App not installed"
**Solution:**
- Check storage space (need 100 MB free)
- Enable "Unknown sources" for APK
- Restart device and try again

**Issue:** "App keeps stopping"
**Solution:**
- Update Android OS
- Clear app cache
- Reinstall app

### iOS

**Issue:** "Cannot download app"
**Solution:**
- Check storage space (need 100 MB free)
- Sign in to App Store
- Check internet connection
- Restart device

**Issue:** "Untrusted developer"
**Solution:**
- Settings → General → Device Management
- Trust the developer certificate

### Windows

**Issue:** "Windows protected your PC"
**Solution:**
- Click **"More info"**
- Click **"Run anyway"**
- (App is safe, Windows is just cautious)

**Issue:** "Installer won't run"
**Solution:**
- Right-click installer → "Run as administrator"
- Disable antivirus temporarily
- Re-download if corrupted

### Mac

**Issue:** "App can't be opened"
**Solution:**
- Right-click app → **"Open"**
- Click **"Open"** in dialog
- Or: Settings → Security → Allow app

**Issue:** "Damaged and can't be opened"
**Solution:**
```bash
# Open Terminal and run:
xattr -cr /Applications/MedicalAppointments.app
```
