========================================
  MEDICAL APPOINTMENT SYSTEM
  Clinic Installation Package
  Version 1.0 - October 31, 2025
========================================

WHAT'S INCLUDED:
----------------
This package contains everything needed to install
the Medical Appointment System on clinic computers.

FILES IN THIS PACKAGE:
----------------------
📄 README_INSTALLATION.md  - Complete installation guide
📄 QUICK_START.txt          - Quick reference guide
📄 INSTALLATION_CHECKLIST.txt - Installation checklist
📄 CONFIGURATION_GUIDE.md   - Configuration instructions
📄 TROUBLESHOOTING.md       - Common issues and solutions

🔧 INSTALLATION SCRIPTS:
-----------------------
  INSTALL_CLINIC.bat         - Full automatic installation
  INSTALL_POSTGRESQL.bat     - Database installation
  INSTALL_NODEJS.bat         - Node.js installation
  SETUP_BACKEND.bat          - Backend server setup
  SETUP_DATABASE.bat         - Database setup
  INSTALL_APP.bat            - Application installation

🚀 LAUNCH SCRIPTS:
-----------------
  START_BACKEND.bat          - Start backend server
  START_APP.bat              - Launch application

✅ VERIFICATION:
--------------
  CHECK_INSTALLATION.bat     - Verify installation

QUICK START:
------------
1. Double-click: INSTALL_CLINIC.bat
2. Wait for installation (10-15 minutes)
3. Double-click: START_BACKEND.bat
4. Double-click: START_APP.bat
5. Login and start using!

REQUIREMENTS:
-------------
- Windows 10 or 11 (64-bit)
- 4 GB RAM (8 GB recommended)
- 2 GB free disk space
- Internet connection
- Administrator privileges

INSTALLATION TIME:
-----------------
Approximately 10-15 minutes

SUPPORT:
--------
For detailed instructions: README_INSTALLATION.md
For troubleshooting: TROUBLESHOOTING.md
For configuration: CONFIGURATION_GUIDE.md
